### 思路

与26题==升序数组==不同，此题是一个==升序链表==

要求删除重复值，返回修改后的链表

### 代码

```python
class Solution:
    def deleteDuplicates(self, head):
        if not head or not head.next: return head
        val_set = set()
        val_set.add(head.val)
        root = ListNode(0)
        root.next = head
        while head and head.next:
            if head.next.val in val_set:
                head.next = head.next.next
            else:
                head = head.next
                val_set.add(head.val)
        return root.next
```
